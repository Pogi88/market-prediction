**Description:

This app is created for the purpose of practicing the use of Git and GitHub. This is a mock stock market prediction app.

The Stock Prediction Calculator is a web application designed for predicting future stock values. 
Users can anticipate the value of a stock by entering its current symbol (e.g., AAPL) and value. 
The application then simulates the projected stock value for the next year, considering a random fluctuation.


**Installation: 

To set up the project locally, you need to follow these steps:

1. Clone the repository:

git clone https://github.com/Pogi88/market-prediction.git
git

2. Navigate to the project directory:

cd market-prediction

**Usage: 

The project structure is the following:

1. index.html: Main HTML page of the web application
2. style.css: CSS file containing the styling for the app
3. script.js: JavaScript file containing the prediction algorithm
4. appinfo.txt: Information about the application
5. readme.txt: README file, further information
